## Pour démarrer les TP en C :

- Télécharger l'ensemble du dépôt, ou clonez le dépôt dans un dépôt local.
- Ouvrir le dossier correspondant au TP dans CLion.
- Bien veiller à ce que CLion reconnaisse le projet comme projet CMake, sinon clic-droit sur le fichier ``CMakeLists.txt`` et sélectionner "Load CMake project".
